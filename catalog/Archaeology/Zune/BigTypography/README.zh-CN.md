# Zune Big Typography

Zune 与 Metro 使用超大、轻字重、常为小写的 Segoe 标题作为空间与品牌元素；滚动时标题可部分离屏，让内容取代传统窗口框。

## 状态与所有权

- 状态：proposed / lab / P1
- 现代 owner：controls.big-title（BigTitle）
- license_review：pending

本展项只拥有历史研究、现代化说明和 Gallery 演示配置，不新增稳定 API、不复制历史素材。
