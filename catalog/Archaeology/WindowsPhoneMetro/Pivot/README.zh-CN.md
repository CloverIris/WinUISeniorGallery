# Pivot

Windows Phone 7/8 的轻量顶层导航：超出屏幕的文字标题横排，当前标题与内容同步平移，左右轻扫连续切换相邻页。

## 状态与所有权

- 状态：proposed / lab / P1
- 现代 owner：controls.pivot-view（PivotView）
- license_review：pending

本展项只拥有历史研究、现代化说明和 Gallery 演示配置，不新增稳定 API、不复制历史素材。
