# Live Tile

Windows Phone 开始屏幕的 Live Tile 以小/中/宽等尺寸显示应用身份与动态摘要，并通过翻转、循环、图标和计数在不打开应用时传递状态。

## 状态与所有权

- 状态：proposed / lab / P1
- 现代 owner：controls.dynamic-tile（DynamicTile）
- license_review：pending

本展项只拥有历史研究、现代化说明和 Gallery 演示配置，不新增稳定 API、不复制历史素材。
