# App Contracts（Windows 8 设计考古）

研究 Windows 8 Search、Share、File Picker、Play To 等系统与应用协作入口，重点是“由用户触发、宿主声明能力、上下文可交接”。本展项不拥有 Provider、权限或平台调用；状态 `proposed` / `lab` / `P2`，许可 `pending`，路由 `/archaeology/windows-8/app-contracts`。`SPEC`、`DESIGN`、`INTEGRATION`、`ACCEPTANCE`、`SOURCES` 分别描述历史协议、现代边界、展示集成、验证和证据；未具备指定现代 owner 时不得实现。
