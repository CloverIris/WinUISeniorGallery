# App Contracts Design Archaeology and Modernization

Retain explicit user intent, disclosure of what will be shared before invocation, and preserved context on return. Reject implicit background sharing, cross-app data probing, and system bars replacing in-app confirmation. Narrow windows use a sequence diagram; wide windows show message flow. Theme/High Contrast make status/error more than color; RTL follows logical direction. Keyboard follows intent/context/result; Reduced Motion omits connection animation; Narrator announces data category and “not actually sent.”
