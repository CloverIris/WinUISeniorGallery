# Snapped View Research Exhibit Integration

This page reads and links only the public specification of `windowing.dock-layout-preview`; it does not call AppWindow, window migration, Shell, or platform permissions. Demo model is page-local read-only data and changes on the UI thread; unload cancels simulated drag/motion and window close retains no state. Missing capability, rejection, exception, or unavailable backdrop produces local explanation/diagnostics—no network, telemetry, or user files.

Use Gallery theme resources and self-made geometry. Windows screenshots, icons, wallpaper, and extracted assets are prohibited; evidence links must disclose leaving the app.
