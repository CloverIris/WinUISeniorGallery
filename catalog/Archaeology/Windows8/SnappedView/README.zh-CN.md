# Snapped View（Windows 8 设计考古）

研究 Windows 8 的应用分屏/吸附布局：用户把任务压缩成并列区域，而非在窗口之间来回切换。它是 `windowing.dock-layout-preview` 的历史研究输入，不拥有窗口管理 API。状态为 `proposed` / `lab` / `P2`，许可审查 `pending`；Gallery 路由为 `/archaeology/windows-8/snapped-view`，只展示结构、约束和现代映射。

阅读 `SPEC` 了解历史区域与状态，`DESIGN` 了解保留/废弃取舍，`INTEGRATION` 了解不越过宿主窗口的界线，`ACCEPTANCE` 了解研究展示验证，`SOURCES` 记录证据与资产禁令。未经现代 owner 独立规格，不得实现。
