# Semantic Zoom (Windows 8 Design Archaeology)

This exhibit studies Windows 8 moving from a detailed collection to a grouped overview, relocating, and returning to detail. Modern owner is `controls.semantic-zoom-view`; this folder owns no control, gesture, or public API. Status `proposed` / `lab` / `P2`, license `pending`, route `/archaeology/windows-8/semantic-zoom`.

`SPEC` explains the historical two-level model; `DESIGN` records semantic hierarchy rather than visual zoom; `INTEGRATION` bounds Gallery demo; `ACCEPTANCE` defines accessibility tests; `SOURCES` retains evidence and asset limits. No implementation before the owner has an independent ready specification.
