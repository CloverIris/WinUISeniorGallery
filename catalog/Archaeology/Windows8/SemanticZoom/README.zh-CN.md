# Semantic Zoom（Windows 8 设计考古）

本展项研究 Windows 8 从详细集合视图缩放到分组概览、再定位并回到细节的交互。现代 owner 是 `controls.semantic-zoom-view`；本目录不拥有控件、缩放手势或公开 API。状态 `proposed` / `lab` / `P2`，许可 `pending`，研究路由 `/archaeology/windows-8/semantic-zoom`。

`SPEC` 说明历史双层模型；`DESIGN` 记录语义层级而非纯视觉缩放；`INTEGRATION` 限制 Gallery 演示；`ACCEPTANCE` 定义可访问性测试；`SOURCES` 保存出处和资产限制。未经 owner 独立 ready 规格不得实现。
