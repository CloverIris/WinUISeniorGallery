# Semantic Zoom Research Exhibit Integration

Gallery uses only a local read-only sample collection and a `controls.semantic-zoom-view` documentation link; no control implementation, network data, user search history, or telemetry. Unload cancels simulated transitions; data replacement resets to readable explanation on the UI thread; missing owner, touch, or capability only falls back. Use theme resources and self-made geometry; Windows screenshots, icons, and extracted assets are prohibited.
