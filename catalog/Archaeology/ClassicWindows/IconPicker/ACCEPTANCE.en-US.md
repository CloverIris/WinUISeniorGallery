## Experience Scenario

10,000 icons realize viewport items only; name/alias search returns stable order; when Segoe Fluent Icons is absent on Windows 10, use the host catalog fallback without downloading or extracting fonts.

## Input, Accessibility, and Performance

The virtualized grid supports arrows, Home/End, Page, and text search; each icon exposes name, category, mirroring, and source rather than only a Unicode value. Light, Dark, High Contrast, Reduced Motion, RTL, 100%–300% DPI, and 200% text scaling pass. Closing leaks no Provider, event, timer, or window.
