# Voice Access Overlay

Windows 11 Voice Access 提供屏幕顶部状态栏、实时命令反馈、可说内容提示，以及“显示数字/显示网格”覆盖层，让用户用语音定位并操作屏幕。

## 状态与所有权

- 状态：proposed / lab
- 现代 owner：未分配（无稳定 API）
- license_review：pending

考古项只拥有研究、现代化说明和 Gallery 配置，不新增稳定 API、不模拟 Windows Shell。
