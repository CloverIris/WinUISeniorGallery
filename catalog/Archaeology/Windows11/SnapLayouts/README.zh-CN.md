# Snap Layouts

Windows 11 自 2021 首发版把 Snap Layouts 放入最大化按钮悬停/Win+Z 入口，以若干分区图示让用户选择窗口落位，并把已贴靠窗口组成 Snap Group。

## 状态与所有权

- 状态：proposed / lab
- 现代 owner：windowing.dock-layout-preview（DockLayoutPreview）
- license_review：pending

考古项只拥有研究、现代化说明和 Gallery 配置，不新增稳定 API、不模拟 Windows Shell。
