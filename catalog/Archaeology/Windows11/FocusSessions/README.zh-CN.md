# Focus Sessions

Windows 11 Clock 的 Focus Sessions 把专注/休息计时、每日进度、Microsoft To Do 任务与可选 Spotify 音乐整合为一个可启动的专注流程。

## 状态与所有权

- 现代 owner：experiences.focus-session（FocusSession）
- license_review：pending

考古项只拥有研究、现代化说明和 Gallery 配置，不新增稳定 API、不模拟 Windows Shell。
