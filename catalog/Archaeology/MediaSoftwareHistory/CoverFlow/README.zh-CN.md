# Cover Flow

Cover Flow 并非 Zune 原创。Apple 于 2006-09-12 在 iTunes 7 中把 Cover Flow 列为新功能：中央封面正对用户，邻近封面以透视角排列，滚轮/拖动/键盘改变中心项。

## 状态与所有权

- 状态：proposed / lab / P1
- 现代 owner：controls.carousel-view（CarouselView）
- license_review：pending

本展项只拥有研究、现代化说明和 Gallery 配置，不新增稳定 API、不提交历史素材。
