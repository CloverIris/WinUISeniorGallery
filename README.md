# WinUI 3 Senior Gallery

[简体中文](README.zh-CN.md) · [English](README.en-US.md) · [PRD](docs/PRD.zh-CN.md) · [Feature Catalog](docs/roadmap/FEATURE-CATALOG.md)

Specification-first advanced controls and modern reconstructions of classic Windows experiences for WinUI 3.

The repository currently contains a buildable .NET 10 / Windows App SDK 2.2 MSIX Gallery scaffold and a governed Catalog. Only `ready` work items may be implemented; Chinese specifications are normative.
